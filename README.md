# Drip Shopping App

A high-end, premium e-commerce shopping experience.

## Project Structure

- **src/components**: Reusable UI components divided into `common` and `admin`.
- **src/contexts**: React Contexts for global state management (Auth, Cart, Wishlist, etc.).
- **src/hooks**: Custom React hooks.
- **src/pages**: Application pages including the Admin Panel.
- **src/reducers**: Redux-style reducers for complex state logic.
- **src/types**: TypeScript interfaces and types.
- **src/utils**: Constant values and helper functions.

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`

## Technologies Used

- React
- TypeScript
- Vite
- CSS (Vanilla)
